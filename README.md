# Exercicios-Javascript

Olá Trainees! fico feliz de ver voces por aqui, em fim, o fim desse repositório é proporcionar um ambiente para que voces resolvam exercicios de programação relacionados a linguagem que usamos na byron, a que usamos de verdade mesmo é o Typescript, que é nada mais que o Javascript com alguns extras, portanto, voces irão treinar com o Javascipt, que é a base.

# Como funciona

Primeiramente, vocês precisam ter o node intalado no computador de voces, isso já deve ter sido ensinado na capacitação até o momento, mas por via das duvidas, segue o link abaixo:

- [Node.js](https://nodejs.org/)

## Instalação

Ao entrar pelo link voces já devem ter recebido automaticamente o fork do repo incial, para verificar, basta ver que no nome desse repo, tem o seu nome de usuario do github logo depois, portanto, peguem o link clicando em code ali em cima, pode ser o de ssh (se já estiver configurado) ou o http e então, crie uma pasta na sua area de trabalho com o nome que desejar

Dentro dessa pasta, abra o git bash e execute o comando abaixo
```bash
git clone COLE O LINK COPIADO AQUI   
```

Depois faça:
```bash
 cd NOME DO REPOSITORIO
```

Então por fim use o comando 
```bash
npm install
```

Uma vez isso feito, voce estará pronto para começar a fazer os exercicios.

## Fazendo os exercicios

Primeiramente, todos os exercicios estão na pasta de atividades, e dentro de cada pasta de tópico haverá três exercicios, um facil, um medio e um dificil.
Você deve apenas editar o arquivo de atividade.js e apenas nos locai especificados, cada exercicio tem um numero associado que está na primeira linha de cada exercicio.

### Testando as respostas

Uma vez que voce tiver feito o codigo de um exercicio, use o comando abaixo para rodar a bateria de testes:
```bash
npm run test:NUMERO DO EXERCICIO
```
Por exempo se fez o exercicio 1 use npm run test:01 e assim em diante
Você pode tambem fazer a correção de todos os exercicios usando o comando

```bash
npm run resumo
```

Boa pratica!!
